## 技术栈

网站用到的技术主要包括：

- Meta Framework 与主要模板语言: [Astro](https://astro.build/)
- CSS Framework: [Tailwind CSS](https://tailwindcss.com/)
- 动态元素：[React](https://react.dev/)
- Markdown 工具链：[unified.js](https://unifiedjs.com/)

后端（主要用于评论系统）：

- HTTP Router: [Hono](https://hono.dev/)
- ORM: [Drizzle](https://drizzle.team/)
- Auth: [BetterAuth](https://betterauth.dev/)
- Database: [libsql](https://github.com/tursodatabase/libsql)

另一些值得一提的使用的库和工具：

- 网页字体分割：[cn-font-split](https://github.com/KonghaYao/cn-font-split)
- OpenGraph 图片生成：[satori](https://github.com/vercel/satori)
- React 动画库：[Motion](https://motion.dev/)

这些都是开源项目，感谢它们的作者和维护者。

## 界面设计

网站的主要设计参考的是 [Void Zero](https://voidzero.dev) 的落地页。

评论区和一些动效设计参考了 [Shiro](https://github.com/innei/Shiro)。
